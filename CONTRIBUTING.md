# Contributing

Thanks for contributing.

## Licensing model

This repository uses dual licensing:

- Code and app infrastructure: Apache-2.0 (see LICENSE)
- Educational question/study content: CC BY-NC-SA 4.0 (see LICENSE-CONTENT)

By submitting a contribution, you agree to license it under the applicable repository license for that content type.

## Source and originality requirements

You must only submit content that is one of the following:

- Your original work, or
- Material you have the right to contribute under compatible terms

Do not submit:

- Real or leaked certification exam questions
- Protected or NDA-restricted test content
- Copyrighted material copied without permission

Maintainers may reject or remove contributions that violate these rules.

## Pull request checklist

- Explain what changed and why
- Include source references for technical claims in question content
- Confirm your submission does not reproduce protected exam content
- Run local checks when applicable (`npm run lint`, `npm test`, `npm run build`)
